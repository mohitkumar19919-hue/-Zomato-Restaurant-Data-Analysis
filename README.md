
# 🍽️ Zomato Restaurant Data Analysis

## 📌 Project Overview

This project analyzes Zomato restaurant data to uncover insights into restaurant ratings, cuisines, pricing, online ordering trends, and customer preferences. The analysis includes data cleaning, exploratory data analysis (EDA), and interactive visualizations.

## 🎯 Objectives

*  Analyze restaurant ratings and customer preferences
*  Identify popular cuisines and restaurant categories
*  Explore the relationship between pricing and ratings
*  Examine online ordering and table booking trends
*  Generate actionable business insights

## 🛠️ Tools & Technologies

*  Python
*  Pandas
*  NumPy
*  Matplotlib
*  Seaborn
*  Plotly

## 🔍 Key Insights

*  Identified top-performing restaurants and cuisines
*  Analyzed the impact of cost on customer ratings
*  Explored location-wise restaurant trends
*  Built visualizations to support business decisions

## 🚀 Outcomes

This project demonstrates skills in data cleaning, exploratory data analysis, data visualization, and business intelligence using real-world restaurant data.
